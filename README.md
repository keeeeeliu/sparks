# creativeApp — AI Portfolio Workspace

A workspace for building a focused AI-engineering portfolio, anchored on a domain I care about: **experiential event planning + zero-proof (non-alcoholic) bartending / aesthetic experience design.**

Goal: land a **Full-Stack AI Engineer / AI Agent Engineer** role, while building things that are meaningful, beautiful, and could seed a future creative business.

## Working principles (from the guideline docs)
- **Build > watch.** Code that runs (and breaks, and you fix) is the only real progress.
- **Deploy early, push to GitHub from day 1.** The commit history is itself a hiring signal.
- **Evals + observability are the differentiator** — the part most portfolios skip.
- **Aim for a few real users**, not just a deployed demo.
- **Aesthetics matter** — make the UIs genuinely beautiful.
- Keep projects **free / non-commercial** until visa work-authorization status clearly permits monetizing.
